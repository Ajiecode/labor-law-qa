# Dify 工作流配置步骤（图文版）

## 第一步：创建工作流应用

1. 登录 Dify → 「工作室」
2. 点击「创建应用」
3. 选择「工作流」→「创建空白工作流」
4. 输入名称：「劳动法问答」
5. 点击「创建」

---

## 第二步：配置「开始」节点

1. 点击画布上的「开始」节点
2. 右侧点击「添加变量」
3. 设置：
   - 变量名称：`query`
   - 类型：文本
   - 显示名称：用户问题
   - 必填：开启

---

## 第三步：添加「知识检索」节点

1. 点击节点之间的「+」或「添加节点」
2. 选择「知识检索」
3. 点击该节点，在右侧设置：
   - 知识库：点击「添加」→ 选择「劳动合同法」
   - Top K：5
   - 相似度阈值：0.7（可选）

---

## 第四步：配置「LLM」节点

1. 点击知识检索节点后面的「+」
2. 选择「LLM」大语言模型
3. 右侧设置：
   - 模型：选择你配置的模型（如 GPT-4 / DeepSeek）
   - 系统提示词：复制下方内容

### 系统提示词（复制使用）

```
你是一个专业的劳动法律咨询助手，专门帮助用户解答关于《中华人民共和国劳动合同法》的问题。

回答要求：
1. 只基于提供的知识库内容回答，不要编造法律条文
2. 回答要准确、清晰、简洁
3. 如果问题与劳动合同法无关，礼貌地说明无法回答
4. 引用相关法条时，说明具体条款编号
5. 适当使用表格或列表使回答更清晰
6. 对于复杂问题，先给出结论，再解释原因
```

4. 上下文变量：选择 {{knowledge_retrieval.result}}
5. 提示词变量：输入 {{start.query}}

---

## 第五步：配置「结束」节点

1. 点击 LLM 节点后面的「+」
2. 选择「结束」
3. 右侧设置输出变量：
   - 变量名称：`answer`
   - 变量值：{{llm.text}}

---

## 第六步：连接节点

确保节点按顺序连接：
```
[开始] → [知识检索] → [LLM] → [结束]
```

---

## 第七步：测试

1. 点击右上角「运行」
2. 在「用户问题」输入框输入问题
3. 点击「运行」查看结果

示例问题：
- 试用期最长多久？
- 签订无固定期限合同的条件？
- 解除劳动合同需要给补偿金吗？

---

## 第八步：发布

1. 点击右上角「发布」
2. 点击「发布更新」
3. 点击「访问API」获取 App ID 和 API Key

---

## 配置完成后的界面

```
┌─────────┐    ┌──────────────┐    ┌─────┐    ┌──────┐
│  开始   │───▶│  知识检索    │───▶│ LLM │───▶│ 结束 │
│ query   │    │ 劳动合同法  │    │     │    │answer│
└─────────┘    └──────────────┘    └─────┘    └──────┘
```

配置完成后，在浏览器打开 `index.html`，填入 API Key 和 App ID 即可开始使用！
